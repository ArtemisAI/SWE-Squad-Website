# Agents

Agents are the core execution units in SWE-Squad.

## Agent Types

- **Explorer** — Fast read-only search agent
- **Planner** — Architecture and design agent
- **Coder** — Implementation agent
- **Reviewer** — Code review agent

## Configuration

Each agent can be configured with specific model parameters and tool access.
